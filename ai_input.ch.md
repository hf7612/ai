# 输入法状态听觉感知自动提示方案

## 项目简介
日常使用电脑时，我们经常需要中英文混合输入，很多时候无暇注视屏幕，稍有分心就无法分辨当前处于中文还是英文输入法状态，极易出现输错字符、反复删除修改的情况，严重影响输入体验与效率。

基于日常使用痛点，本项目实现一套**后台无感监听 + 听觉状态区分**的轻量化解决方案：
切换至中文输入法时关闭按键音效保持静音，切换至英文输入法时自动开启复古按键音效，仅凭声音即可快速判断当前输入法状态，实现盲打精准区分。
